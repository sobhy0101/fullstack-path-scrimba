### How to Test

1. **Generate Schemes**: Click "Get Color Scheme" or press `Enter`
2. **Copy Colors**: Click any color card or press `C` on focused card
3. **Change Format**: Click HEX/RGB/HSL buttons or press `1`, `2`, `3`
4. **Random Color**: Click "Random" button or press `R`
5. **Share URL**: Click "Share URL" to copy shareable link
6. **Keyboard Nav**: Use arrow keys to navigate between colors

### Keyboard Shortcuts

- `Enter` - Generate color scheme
- `C` - Copy focused color
- `R` - Randomize seed color
- `1` / `2` / `3` - Switch to HEX / RGB / HSL
- `Arrow Keys` - Navigate between colors
- `Home` / `End` - First / Last color
